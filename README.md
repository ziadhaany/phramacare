# PharmaCare

PharmaCare is a Pharmacy Management System developed to simplify and automate daily pharmacy operations. The system helps manage medicines, prescriptions, sales, customers, and inventory through a centralized web-based platform. The project was built with a focus on usability, data accuracy, and efficient workflow management.

## Overview

As the developer of PharmaCare, I designed and implemented a complete pharmacy management solution that supports core pharmacy activities. The system was developed to reduce manual work, improve inventory tracking, and provide an organized environment for managing pharmaceutical operations.

Throughout this project, I worked on database design, backend development, frontend implementation, and system integration to deliver a functional and maintainable application.
